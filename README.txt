Utilisez la commande "make" pour compiler tous les fichiers nécessaires et créer des fichiers exécutables.
Utiliser la commande "make clean" pour decompiler.
Jusqu'à TP5 donc pour la partie Calculette on execute nos tests avec ./calculette avec argument entree.txt, à partir de TP5 on utilise ./interpreteur avec argument entree.txt.
Pour utiliser test_lexeme on execute la commande ./test_lexeme avec argument entree.txt.

Manuel utilisateur se trouve dans le fichier doc.
